## Summary

The remaining free-running trigger effects now declare `timebase: 'voice'`, so the generator
bridge feeds them a **hit-relative clock** — they start from their origin phase on the hit and
**restart on retrigger** instead of unmasking a free-running animation mid-cycle. Zero
generator-signature changes; every conversion is a one-line declaration (the bridge swaps
`ctx.timeMs` / `ctx.transport`, per S25).

- **Tier 1 — 9 runtime conversions** (read a phase clock; the flag swaps it): `synced-hoops`,
  `strobe`, `starfield`, `collisions`, `sacred-hogs`, `gravity-wells`, `orbit-rings`,
  `comet-trails`, `temp-sweep`. The 5 stateful ones were audited: per-voice `genState` already
  resets on (re)spawn, so seeds/accumulators restart from t=0 and never leak across voices.
- **Tier 2 — 7 byte-parity declarations** (already read `trig.ageMs` directly, so hit-relative
  by construction): `radial-wash`, `wave-collapse`, `whole-drum`, `whole-kit`, `follow-hoop`,
  `burst`, `lightning`. Flagging them `voice` is byte-parity (they read no `ctx.timeMs` /
  `ctx.transport`) and makes `gen.timebase` the **complete** source of truth S27's thumbnail
  renderer reads.
- **Base/ambient stays absolute**: `breathing-kit`, `hue-rotate-kit`, `solid-base` untouched.
- **Textures stay absolute** (single per-effect flag can't be voice-when-hosted +
  absolute-when-a-look; looks must not phase-snap on recall — S25 recipe pt 3).
- **Deliverable**: the 41-effect timebase audit at `docs/handoff/rock-solid/effect-timebase-audit.md`.

## Context pack (brief)

- The conversion recipe is unchanged from S25 — the seam is still `generator-bridge.ts:115`
  (core) / `render.ts:221` (web); both read `(gen.timebase ?? 'absolute') === 'voice'`. This
  slice added **no bridge logic**, only registry flags, so web parity is preserved by
  construction (the web bridge already handles any voice effect generically).
- **Two tiers of "voice"**: clock-swap (reads `ctx.timeMs`/`ctx.transport`) vs. intrinsic
  (reads `trig.ageMs`). Both are hit-relative; doc 06 §A defines voice as "animate from
  `trig.ageMs`", so both carry the flag.
- **`comet-trails` reads no clock** (only `ctx.dt`) — restart is via `genState` reset, so its
  total accumulated `dt` is birth-independent (excluded from the birth-dependence golden;
  covered by restart + no-leak).
- **For S27**: `gen.timebase === 'voice'` is the complete hit-relative set. Seq-driven effects
  (`confetti-burst`, `pixel-accum`, `colour-melody`, `swing`, `sidechain`) are `absolute` but
  animate via **new-hit seq**, not age — their thumbnails need the synthetic trigger's seq
  bumped. `velocity-flames` is a hybrid (age flame + wall-clock flicker) — loop its trigger age
  regardless of the flag if a frozen flame reads poorly. (Details in the audit's S27 notes.)

## Deviations

- **Textures kept absolute (the one place S26 does not fully satisfy doc 06's wish).** Doc 06 §A
  wants textures "voice when trigger-hosted" but "absolute when used as looks". `timebase` is a
  single per-effect flag and both bridges route every effect (looks included) through the same
  `renderVoice` path, so a texture can't be both. The hard constraint (looks must not phase-snap
  on recall) and the S25 recipe ("Do NOT add the flag to textures used as looks") both point the
  same way → textures stay absolute. A per-instance override (voice-when-hosted-on-a-trigger)
  needs a **voice/bus-level** timebase, not a per-effect flag — out of S26 scope (fits doc 06 §C
  modifiers / doc 10 modulation). Documented in the audit's Decisions §1.
- **Tier 2 declarations go slightly beyond the 9 named conversions** — deliberate. The 7
  intrinsic age-readers are already hit-relative; flagging them `voice` is byte-parity (full
  suite green) and makes `gen.timebase` a complete contract for S27 (assignment: "S27 READS the
  registry timebase flags you set"). Trivially reversible if unwanted (7 one-line declarations).
- **`synced-hoops` (category `base`) converted per spec** — it is named in the conversion list in
  both the assignment and slice spec, and is NOT in the enumerated stay-absolute set
  (breathing-kit / hue-rotate-kit / solid-base). Category is a palette grouping, not a timebase
  directive.
- **`helix` / `wipe-3d` left absolute** — free-running washes not in the named set; left absolute
  for scope discipline (noted as follow-up candidates; `wipe-3d` mirrors `orbit-rings`, which IS
  named + converted — asymmetry documented).
- **No generator body edits** — every conversion was flag-only, as the S25 recipe predicted.
