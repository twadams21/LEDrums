## Summary

  resolved library refs as `LibraryRefRow`) and **Song Library** (`LibrarySongRow` pool).
- **Row actions**: local song → *Add to library* (`exportSongToLibrary`, tick-flash); pool → *Import to
  show* (`importSongReference`, disabled + "In this show" badge once referenced); ref → *Detach copy*

  naming the block reason (mirrors the store guard via pure `librarySongRows.deletable`).
- **Naming**: pool = "Song Library", show group = "This show"; sections view already reads "Setlist". No
  "Show Song Setlist manager" remnants existed (grep-verified).
- Pure view-models `showSongRows`/`librarySongRows` in `objects-view.ts` keep the .svelte thin.
