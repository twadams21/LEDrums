## Context pack for S47/S48/S49
**Interface** (`import { PixliteClient, ... } from '@ledrums/io'`), host-bound, all `Promise`:
- `probe(timeoutMs?) → ControllerIdentity | null` — `{ host, prodName, nickname, fwVer, apiVer[], authReqd }`
- `statisticRead(paths: string[]) → ControllerStats` — `{ universes:{ sACN:UniverseRx[]; artNet:UniverseRx[] }, rates:{ inFrmRate?, outFrmRate? }, health:{ tempC?, bankVoltsMv?, portStatus?, ethLinkUp? }, raw }`. `UniverseRx = { uniNum, timedOut, inGood, inBadSeq, inLowPri?, priority?, sourceName? }`. **`timedOut === false` = RECEIVING.**
- `identify(durationS)` (0 off, 1–120 s, 121 continuous) · `modeTestData(params)`
**Real:** `new HttpPixliteClient({ host, port?=80, apiVersion?='v1.7', user?='admin', password?|auth?, timeoutMs?=2000, transport? })`. Pass `auth` (the Base64URL hash) NOT plaintext — matches "store hash on Project". `transport` is injectable for io's own tests; prod uses the default.
**Fake (inject in server tests):** `new FakePixliteClient({ host?, identity?, stats?, delayMs? })` — mutate `.identity`/`.stats`, set `.failNext=Error` (clears after one call); asserts via `.calls[]`, `.identifyCalls[]`, `.testDataCalls[]`. Defaults: `defaultIdentity()`, `defaultStats()` (3 sACN universes receiving, healthy). Non-PixLite host: `{ identity: null }`.
**Discovery:** `sweep(cidrOrAddrs, prober, { timeoutMs?=400, concurrency?=32 }) → RankedCandidate[]` (best-first; PixLite-branded rank above generic). Prod prober: `(h,t) => probe(h, t, { transport, port })`. `expandCidr`/`expandTargets` cap at 1024 hosts. Sweep is standalone (unqueued) — sequential rule is per-controller only.
**Gotchas the server MUST respect:** one client instance per controller (its queue enforces sequential); poll `statisticRead` at 1–2s only while a client has Monitor/Patch open (no idle traffic); errors surface as `PixliteError{code,msg}` or `Error` (HTTP non-200 / transport) — treat both as "controller unreachable/lost" for lastSeen aging.
