# S48 — Controller panel UI (Group L, issue #56)

## Summary
Extended the S03 output status panel in place with the adopted PixLite's own truth — completing the
confidence chain (app fired → server sent → **controller received → outputting**). New pure composite
`ControllerStatusPanel` (identity, per-universe rx, rates, health, discovery + Adopt-IP/Identify/Discover)
renders below the fault row via new opt-in props on `OutputStatusPanel`; derivations live in
`output-status.ts`. Store gained `controllerStatus`/`controllerCandidates` state, the two server
broadcasts, and the `watch/discover/adopt/identify` send helpers. Did **not** fork S03.
