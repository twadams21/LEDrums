## Context pack for S48 (the panel)

**`controllerStatus` message** = `{ t:'controllerStatus'; status: ControllerStatus | null }` (null =
nothing adopted). `ControllerStatus` (in `@ledrums/protocol`), every field the panel renders:
- `host` — adopted IP (stable key).
- `reachable` — false = poll timed out/errored → show LOST; `lastSeen` stops advancing.
- `identity` — `{ host, prodName, nickname, fwVer, authReqd } | null`. Panel: nickname=name,
  prodName=model, fwVer=firmware, host=IP. null only in the gap before the first probe.
- `universes` — `ControllerUniverseRx[]`: `{ uniNum, protocol:'sACN'|'artNet', receiving, inGood,
  inBadSeq, inLowPri?, priority?, sourceName? }`. **`receiving` is the headline** (= controller's
  `!timedOut`); `receiving:false` is the "not receiving" state to make unmissable.
- `rates` — `{ inFrmRate?, outFrmRate? }` (Hz). `health` — `{ tempC?, bankVoltsMv?, portStatus?, ethLinkUp? }`.
- `lastSeen` — epoch ms of last success (null=never). `Date.now()-lastSeen` = quiet duration.

**Discovery** — send `{ t:'discoverControllers' }`; server replies `{ t:'controllerDiscovery';
candidates: DiscoveredController[] }` (best-first, replaces the list; empty = none found).
`DiscoveredController = { host, prodName, nickname, fwVer, authReqd, score }`.

**Actions** — `{ t:'adoptController'; host }` (probe+persist; failure → `error` reply), `{ t:'identifyController'; durationS }` (flash LED). Both editor-gated. Adopt-IP → also `setOutput{ host }`.

**Subscribe** — send `{ t:'watchController'; watching:true }` on panel open, `false` on close (disconnect
auto-clears). This is what signals "Monitor/Patch open" and the only thing that starts polling; the
server broadcasts the current status immediately on watch. Adopted state lives at
`project.controller = { host, nickname, auth? }` (on the `state` message's `project`).
