# S08 — Boot overlay + shell reduction + share gating

**Branch:** `slice/S08` (on `group/C` via S06) · **Commits:** `1a89999`, `71ba66e`, `eaf7586`

## Summary
Reduced the native shell to a dumb bootstrap (spinner + fatal error), moved the share/update
surface into the web app, added a token-styled desktop boot overlay driven by the S06
`desktopBridge`, and gated `ShareInfo` on the running stage. Shell colours now inject from
`tokens.css` at bundle time so the shell can't drift from the design system.
